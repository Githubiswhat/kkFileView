# 关于本文档

## 概述

本文档提供了Vastbase G100数据库常用的管理员操作，包括集群管理、备份恢复、例行维护等内容，方便管理员更好地使用数据库。

## 读者对象

适用于所有使用Vastbase G100数据库的管理员。

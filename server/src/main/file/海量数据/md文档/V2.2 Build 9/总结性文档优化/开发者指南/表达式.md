# 表达式

本章主要介绍SQL表达式。

SQL 表达式是构成 SQL 语句的全部或一部分的字符串。 表达式是计算值的一个或多个值，运算符和SQL函数的组合，它们计算结果为确定的值。 这些SQL 表达式就像公式，用户可以使用它们在数据库中查询特定的数据集。

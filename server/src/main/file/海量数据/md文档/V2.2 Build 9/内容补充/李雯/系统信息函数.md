- has_cek_privilege(user, cek, privilege)

描述：指定用户是否有访问列加密密钥CEK的权限。参数说明如下（3个入参）：

| 参数名    | 合法入参类型 | 描述       | 取值范围                                                     |
| :-------- | :----------- | :--------- | :----------------------------------------------------------- |
| user      | name，oid    | 用户       | 用户名字或id。                                               |
| cek       | text，oid    | 列加密密钥 | 列加密密钥名称或id。                                         |
| privilege | text         | 权限       | USAGE：允许使用指定列加密密钥。<br />DROP：允许删除指定列加密密钥。 |

返回类型：Boolean

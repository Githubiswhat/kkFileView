# 数组类型

本文列出了可用的数组类型：

| 名称              | 描述                                | 范围       |
| ----------------- | ----------------------------------- | ---------- |
| oidvector         | oid数组，在系统表中使用             | 变长字符串 |
| int2vector        | int2数组，在系统表中使用            | 变长字符串 |
| oidvector_extend  | oid数组，用于系统表并支持toast存储  | 变长字符串 |
| int2vector_extend | int2数组，用于系统表并支持toast存储 | 变长字符串 |

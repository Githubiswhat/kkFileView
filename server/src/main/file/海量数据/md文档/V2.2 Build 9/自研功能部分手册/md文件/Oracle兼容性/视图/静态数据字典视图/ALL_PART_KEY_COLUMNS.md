#### ALL_PART_KEY_COLUMNS

ALL_PART_KEY_COLUMNS视图提供当前用户可访问数据库中的分区表的分区键列的信息。

| 列名            | 类型         | 描述                            |
| --------------- | ------------ | ------------------------------- |
| owner           | text         | 分区表的所有者                  |
| schema_name     | text         | 分区表所属的模式名              |
| name            | text         | 分区名称                        |
| object_type     | character(5) | 仅为支持兼容性。该值始终为table |
| column_name     | text         | 字段名                          |
| column_position | integer      | 分区键中列的位置                |

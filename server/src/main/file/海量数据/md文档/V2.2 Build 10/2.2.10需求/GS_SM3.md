# GS_SM3

**功能描述**

gs_sm3函数用来计算输入字符串的SM3摘要。

**语法格式**

```
gs_sm3(text)
```

**参数说明**

text：输入的字符串。

**注意事项**

- 返回值为SM3哈希值的十六进制字符串，长度固定为64。
- 如果输入data为空或者计算失败，返回结果为null。

**示例**

调用函数。

```
select gs_sm3('abcdef');
```

结果显示如下：

```
                              gs_sm3                              
------------------------------------------------------------------
 5d60e23c9fe29b5e62517e144ad67541c6eb132c8926637b6393fe8d9b62b3bf
(1 row)
```
